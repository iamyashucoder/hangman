# Hangman Game

A simple Python terminal-based Hangman game. The player guesses letters one by one to reveal a hidden word before running out of lives.

## Features

- Random word selection
- Letter-by-letter guessing
- Tracks correct guesses
- Shows hidden word using underscores
- Hangman ASCII art stages
- Win and lose conditions
- Beginner-friendly Python code

## How to Run

Make sure Python is installed on your computer.

Run the game with:

```bash
python hangman.py
